Welcome to ArrayLSTM's documentation!
=====================================

ArrayLSTM provides a pytorch implementation of Recurrent Memory Array Structures by Kamil M Rocki.
This code was implemented as part of the TODO paper.
We ask people to cite both works when using the software for academic research papers.
